SequenceToArrayConverter

com.isencia.passerelle.actor.sequence.SequenceToArrayConverter

Keeps track of all sequences for which messages pass through it. Each message in a sequence is maintained in cache until its complete sequence has passed. Then the sequence is sent out in the form of a message containing an array of all original msg contents. So the msg contents in a sequence had better been of the same type!
