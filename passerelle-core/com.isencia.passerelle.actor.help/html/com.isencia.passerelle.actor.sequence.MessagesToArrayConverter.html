MessagesToArrayConverter

com.isencia.passerelle.actor.sequence.MessagesToArrayConverter

Converts a series of incoming into an outgoing message containing an array with the contents of the incoming msgs. The array is generated when a trigger msg is received. Remark that this actor can not guarantee exact array contents in the standard PN domain (or Passerelle's extension of it), as there's no certainty on the order in which data msgs and the trigger may arrive. For tighter control, use message sequences as input together with the SequenceToArrayConverter.
