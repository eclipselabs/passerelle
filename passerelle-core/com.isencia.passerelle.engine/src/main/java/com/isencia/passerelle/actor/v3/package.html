Passerelle 3 provides a new simpler API for actor implementations.

The new API has 2 main goals:

  • Encapsulating the handling of multiple input ports of different types (blocking/non-blocking)
  • Simplifying the actor API 1 main method in most cases, i.o. 3 (preFire/fire/postFire)
